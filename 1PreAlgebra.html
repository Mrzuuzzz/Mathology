Mathology

Pre-Algebra is the category of math that includes: integers, order of operations, algebraic expressions, one and two-step equations, proportions, percents, probability, and linear equations.

- Order of Operations: Commonly known as PEMDAS (Parenthesis, Exponents, Multiplication, Division, Addition, Subtraction) the order of which you do mathematical operations.

- Algebraic Expression & Two-Step Equations: Learn to create and solve algebraic expressions in multiple steps.

- Probability: Learn how probability works and be able to create ratios based on any given number of outcomes.

- Linear Equations: Learn the basics in linear equations, involving slope, and solving equations using substitution, elimation, or graphing.
