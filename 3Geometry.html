Mathology

Geometry is the branch of math that involves different shapes and their angles. How to use them to solve for other angles / unknown values. How shapes relate to each other and visualizing 3D shapes on a graph.

Some topics in Geometry:

- Angles: Learn angles in different polygons, their relations to each other, and how to solve for them in different shapes, such as triangles. Ex. A triangle has 3 angles, each angle must add up to 180 degrees.

- Triangle Similarity: Find out how triangles can be similar, how to identify similar triangles, label them using the correct postulate, and solve for unknown values using properties of similar triangles using ratios.

- Trigonometric Functions: Use trigonometric functions (Sine, Cosine, Tangent) to solve for unknown sides/angles in right triangles.
